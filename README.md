# Cloud-Share-Files
Built a cloud-based file sharing application using React.js, Spring Boot, and MongoDB.

☁️ CloudShare

CloudShare is a full-stack cloud file-sharing platform that enables users to securely upload, manage, and share files online. The application provides secure authentication, file management capabilities, and subscription-based features through payment integration.

🚀 Features
User Authentication and Authorization
Secure Login and Registration
JWT-Based Security
Clerk Authentication Integration
Upload and Manage Files
Responsive User Interface
File Sharing Functionality
Subscription & Payment Integration using Razorpay
RESTful API Architecture
Protected Routes and Secure Resource Access
